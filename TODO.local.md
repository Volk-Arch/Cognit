## 🧪 Тестирование

Стратегия: единый прогон после того как всё собрано.

### Базовые сценарии

- [ ] Smoke: запуск → авто-инит агентов → use → вопрос → ответ
- [ ] Авто-инициализация: agents/ есть, .pkl нет → паттерны создаются при старте
- [ ] Авто-выбор агентов: вопрос про стиль → style активируется автоматически
- [ ] grow_policy: grow-паттерн растёт с диалогом, retrain не сохраняет
      — `~` отображается в промпте и /list для grow-паттернов
      — grow-паттерн пропускается при --refresh-file (не пересоздаётся)
      — `/load ?name @path` форсирует retrain на feature-ветке

### Оркестратор и переключение

- [ ] `python cognit.py` стартует Transformer из .echo.json
- [ ] `expand <задача>` → оркестратор переключает на RWKV без перезапуска
- [ ] `route <задача>` в Transformer → RWKV авто-запускается → файлы возвращаются
- [ ] `route <задача>` в RWKV → handoff на Transformer с файлами
- [ ] expand → RWKV загружается, задача передана → route → Transformer
- [ ] Цепочки агентов: ответ содержит "не хватает контекста" → expand предложен
- [ ] unload_model: VRAM освобождается при переключении (nvidia-smi до/после)

### Git-интеграция

- [ ] post-commit хук: сделать коммит → паттерн изменённого файла пересоздаётся
- [ ] --refresh-file: находит изменённый файл, пересоздаёт только нужные паттерны
- [ ] post-checkout: смена ветки → уведомление если нет RWKV-индекса

### Команды

- [ ] /patch: diff в ответе → применяется с подтверждением → бэкап создан
      — файл определяется из +++ заголовка автоматически
      — `/patch @file` переопределяет целевой файл
      — если diff нет → модель переспрашивается автоматически
- [ ] /agent: `/agent style arch` включает ambient для двух агентов
      — промпт меняется на `[auth + style + arch]>`
      — `/agent off` возвращает обычный промпт
- [ ] /review @file работает эфемерно, паттерн не изменяется после
- [ ] авто-детекция expand/route в ответах модели (RWKV: ответ содержит route → детектор подхватывает)
- [x] /load composite: `/load mix @dir1 @dir2` → единый паттерн из двух источников

### Конфиг

- [x] .echo.json с model_path читается обоими бэкендами
- [x] cognit_setup.py пишет пути к моделям в конфиг

### Оркестратор-агенты
